<details open>
<summary><h3 style="display:inline">🧬 Primitive-based augmentation v2 — 抛弃 optimizer,3 µ choice 统一公式</h3></summary>

> **背景**: 5/15 优化器路径在 9-action 上跑通但很 fragile (per-action 调参 / regularizer mess)。今天彻底重写成 procedural primitives,基于 1995 经典 motion-signal-processing。

**核心公式 (Bruderlin&Williams 1995 displacement mapping)**:
```
dof_aug[t] = μ(t) + k(t) · (dof_seed[t] − μ(t))
```

**3 个 μ choice** (per-action subclass dispatch):

| Subclass | μ choice | Actions |
|---|---|---|
| **A1** contact-periodic | `anchor_traj` (μ(t) lerp through contact valleys) | clap |
| **A2** free-oscillation | `mean_pose` (μ = const = dof.mean) | wave_hand, wave_hands, beckon |
| **B** single-stroke | `first_frame` (μ = const = dof[0]) | point, nod, salute, bow, kick, punch |
| **C** held-pose | `first_frame` | shrug |
| **D** contact-grip | `anchor_traj` | handshake |

**Phase 分段 (Kendon 1994)**: prep / stroke / retract,k 仅在 stroke 应用,prep/retract 保留 seed。

**新文件** (`src/data_augment/`):
- `primitives.py`: P1 (scale_deviation) + P2 (time_warp + time_warp_extend) + P3 (pose_offset) + helpers (build_mu_for_seed, per_cycle_normalize_deviation, enforce_collision_safe, G1_ANATOMICAL_LIMITS)
- `phases.py`: auto_segment_phases (velocity) + detect_valleys_all (hand_dist) + detect_anchors_motion_energy + filter_rhythmic_cluster + kendon_k_schedule
- `taxonomy.py`: ACTION_SUBCLASS + SUBCLASS_MU_CHOICE + ANCHOR_SIGNAL_PER_SUBCLASS + ACTIVE_DOF_PER_SUBCLASS

**新 test script**: `scripts/aug_v2_taxonomy_test.py` — 一行 dispatch (查 taxonomy → 取 μ choice + active dofs + anchor signal → run)。

**决定 / Next**: framework 锁定。下一轮加 LHS sampler + post-hoc normalizer 后即可造数据集。

<details open>
<summary><h3 style="display:inline">🌲 Hierarchical taxonomy — LOCOMOTION ⊥ MOTION + MOTION 5-subclass</h3></summary>

> **背景**: 跟 advisor lab 的 3-tier architecture 一致 — LOCOMOTION → Tier 1.3 (BeyondMimic walker),MOTION → Tier 1.2 (我们这套 + diffusion)。

```
TOP cut: max ‖root_xy_displacement‖ > 50cm → LOCOMOTION; otherwise MOTION

LOCOMOTION (xy 平移显著):
  L1 periodic-gait    walk, jog, run, march
  L2 special          crawl, climb
  L3 directional      turn-while-walking

MOTION (xy 几乎不动):
  A1 contact-periodic  clap, drum            anchor_traj (hand-hand)
  A2 free-oscillation  wave, beckon          mean_pose
  B  single-stroke     point, nod, salute,   first_frame
                       bow, kick, punch
  C  held-pose         shrug                 first_frame
  D  contact-grip      handshake             anchor_traj (hand-hand)
```

**Edge cases (user decided)**:
- kick / jump-in-place / bow → MOTION (xy 不动)
- turn-in-place → MOTION (后期)
- sit_down / stand_up / get_up → 排除(compositional transition)

**Paper framing**: 跟 DMP discrete/rhythmic taxonomy (Saveriano 2023 IJRR) 对齐 — 不是 if-else mess,是 robotics 公认 physical structure。

**决定 / Next**: perceptual_bench 9 actions 全在 MOTION 5 subclass 里。LOCOMOTION 留给后续扩 BABEL 训练。

<a id="d16-anatomical"></a>

<details open>
<summary><h3 style="display:inline">🛡️ Seed-aware anatomical clamp — 防超伸但不破 clap 接触</h3></summary>

> **背景**: G1 机械 joint_limit elbow 允许 [-60°, +97°](允许 hyperextend),knee 类似 [-5°, +165°]。User 发现 wave_hand 胳膊会推到 180° 完全直 — 不自然。

**v1 (global anatomical)**:
```python
G1_ANATOMICAL_LIMITS_LO[elbow_idx] = max(G1_mech_lo, 0.10 rad ≈ 6°)
G1_ANATOMICAL_LIMITS_LO[knee_idx]  = max(G1_mech_lo, 0.10 rad)
# Final clamp: dof_aug = np.clip(dof_aug, ANATOMICAL_LO, ANATOMICAL_HI)
```

**Bug**: clap 接触瞬间 seed R_elbow 实测 -10°(seed 自身略 hyperextend,让手能并拢),被强行推到 +5.7° → 手分开 0.165m,合不上。

**v2 (seed-aware,user 确认 fix)**:
```python
eff_lo[t, d] = min(GLOBAL_ANATOMICAL_LO[d], dof_seed[t, d])
eff_hi[t, d] = max(GLOBAL_ANATOMICAL_HI[d], dof_seed[t, d])
dof_aug = np.clip(dof_aug, eff_lo, eff_hi)
```

逻辑:**output 不会比 seed 更糟,但允许 match seed**。
- seed elbow +80°: floor = +5.7°(anatomical 兜底)
- seed elbow -10°: floor = -10°(允许 match seed 接触姿态)

**验证** (k=1 时 hand_dist_min):

| Action | Before fix | After fix | Seed |
|---|---|---|---|
| clap | 0.165m ❌ | **0.110m** ✓ | 0.110m |
| punch | 0.168m | 0.168m ✓ | 0.168m |
| wave_hand | 0.538m | 0.538m ✓ | 0.538m |

**Universal**: 适用所有 primitive (P1/P2/P3) + 未来 optimizer 路径。

**决定 / Next**: 锁。也写进 optimizer 路径的 default。

</details>

<a id="d16-collision"></a>

<details open>
<summary><h3 style="display:inline">🦴 Full-body collision avoidance — 13 pairs + per-frame lerp enforce</h3></summary>

> **背景**: User 看到 punch k>1 时拳头穿胸。Optimizer 路径有 COLLISION_PAIRS_UPPER_BODY 但 primitive 路径没用。FULL_BODY 之前是 TODO。

**扩展 COLLISION_PAIRS_FULL_BODY** (13 pairs):

| 区域 | Pairs | Min distance |
|---|---|---|
| **上身** (10) | wrist↔torso ×2 (tightened 0.18→0.22) / elbow↔torso ×2 (0.13→0.17) / wrist↔wrist (0.06 允许 clap) / elbow↔elbow / wrist↔own shoulder ×2 / cross-side wrist↔shoulder ×2 (NEW) | 0.06-0.22m |
| **腿-腿** | knee↔knee / ankle↔ankle / hip↔hip | 0.08-0.12m |
| **手-腿** | wrist↔knee 同/异侧 / wrist↔ankle | 0.08-0.12m |
| **肘-膝** | elbow↔knee 同侧 (crouch/kneel) | 0.10m |

**新函数 `enforce_collision_safe`** (Option A: per-frame lerp toward seed):
```python
for t in violating_frames:
    binary-search α ∈ [0.9, 0.8, ..., 0.0]:
        dof_try = α·dof_aug[t] + (1−α)·dof_seed[t]
        if all pairs OK at this α: dof_safe[t] = dof_try; break
```

**严重性**: 高 k 时违反帧 5-30%(punch k=3 时 23%)。处理后 0% 穿模。

**Universal**: 也加进 test script default ON。Clap 不受影响(wrist↔wrist 0.06m 允许接触)。

**决定 / Next**: 默认 ON,所有 primitive 路径。

<summary><h3 style="display:inline">🎚️ Active DOF mask per subclass — 解腿乱摆 bug</h3></summary>

> **背景**: User 反馈 bow / nod / punch k>1 时腿跟着摆动 — 不应该。腿(DOF 0-11) seed 几乎不动,但 P1 等比放大让小 deviation × k 被放大。

**Fix**: taxonomy.py 加 `ACTIVE_DOF_PER_SUBCLASS`:

```python
UPPER_BODY_DOFS = [12..28]  # waist + arms (17)
LEG_DOFS        = [0..11]
ACTIVE_DOF_PER_SUBCLASS = {
  'A1'/'A2'/'B'/'C'/'D': UPPER_BODY_DOFS,  # 上身动作锁腿
  'B-leg':               ALL_DOFS,         # kick 全身 active
}
```

**P1 改造**:
```python
dof_aug[:, mask] = dof_transformed[:, mask]  # active 用 P1
dof_aug[:, ~mask] = dof_seed[:, ~mask]       # inactive 严格 = seed
```

**效果**: 12 个上身 action 腿严格静止,kick 单独保 leg-active。

**决定 / Next**: 锁。

<a id="d16-p2-couple"></a>

<details open>
<summary><h3 style="display:inline">⏱️ P2 time_warp_extend + --time-warp-couple flag</h3></summary>

> **背景**: User 提议 — 想让 A 不变,把动作整体慢下来,保持振幅。理论:s=k 时 A 不变(P1 amp k² × P2 slow 1/s² = 1)。

**新函数 `p2_time_warp_extend(arr, s)`** — output T' = round(T·s) via linear interp。

**CLI flag `--time-warp-couple coef`**:
- coef=0 (default): no warp
- coef=1.0: s=k (理论 A 不变)
- coef=0.5: s=√k (partial)

**实测 (wave_hand)**:

| Couple | k=2 V | k=2 A | T |
|---|---|---|---|
| 0 | +0.37 | **+0.99 saturated** | 90 |
| 0.5 | +0.27 | +0.82 | 127 |
| 1.0 | +0.10 | +0.32 | 180 |

**理论 vs 实际**: 数学对(continuous time),但 discrete interp 把高频信号磨平 + tanh 归一化非线性 → 实际 A 比理论降更多。User 说 V/A 指标晚点调,接受当前 couple=1.0 用来"看 slow-mo"效果。

**决定 / Next**: couple=1.0 用作 slow-mo demo,**A 指标 user 自己调**(可能改成 time-invariant per-second energy)。

<summary><h3 style="display:inline">📖 Lit search — 我们的 framework 在 1995-2023 lineage 里 100% defensible</h3></summary>

> **背景**: User 问 "动画领域都怎么做的能不能找找思路"。派 agent 搜 5 方向。

**核心 finding**: 我们 framework 每个 piece 都有 prior work,**组合方式没人做过**(applied to NMI VAD-conditioned humanoid)。

**Must-cite (methods section)**:
1. **Bruderlin & Williams SIGGRAPH 1995 "Motion Signal Processing"** — displacement mapping 是 P1 直接前身
2. **Witkin & Popović SIGGRAPH 1995 "Motion Warping"** — keyframe constraints (我们 anchor 帧的范式)
3. **Saveriano et al. 2023 IJRR "DMP Tutorial Survey"** — discrete/rhythmic taxonomy 跟我们 A1/B 对应

**Must-differentiate (related work)**:
1. **Aberman SIGGRAPH 2020** — neural style transfer (我们: procedural + 可解释 + 0 data)
2. **RG-VLMD 2025** — monolithic V-A diffusion on NAO (我们: +D + cross-channel handover + N=30 study)

**Disney 12 principles 对齐**: Anticipation = Phase I, Exaggeration = P1 k>1, Follow-through = Phase III, Slow-in/out = boundary fade。Paper 可以说"algorithmic implementation of Disney principles"。

**RG-VLMD competitor analysis (separate task)** 确认了 3 个 reviewer-proof differentiator:
1. V-A-D not V-A (D is novel angle for HRI)
2. Cross-channel handover on G1 vs gesture-only NAO
3. Composable diffusion + N=30 user study vs monolithic + deferred user study

**决定 / Next**: paper §2/§6 写作时直接引这 5 篇。
