>
> **操作**:
> - 解压 `friend_babel_walk_va_g1_2026-05-12.zip` → `friend_babel_walk_va_g1_2026-05-12/walk/` 29652 NPZ
> - 验证 schema:每个 NPZ 含 `dof_pos(T,29)` + `root_pos(T,3)` + `root_quat(T,4)` wxyz + `valence_label(T,)` + `arousal_label(T,)` + `foot_contact(T,2)` + `act_cat='walk'` + BABEL provenance
> - 验证兼容性:DOF 29 ✓ · joint order 跟 G1 default pose 对齐 ✓ · quat 是 wxyz ✓ · fps 20 → 50 走 csv_to_npz lerp/slerp upsample ✓
>
> **结果**: schema 几乎完全兼容,只需要补 `joint_vel` + `body_*_w` (37 bodies) 走 IsaacSim FK 一次。
>
> **决定 / Next**: 写一个 `friend_npz_to_npz.py` 改 `csv_to_npz.py` 的 `_load_motion()` 读 NPZ 即可。其他不动。

<summary><h3 style="display:inline">⭐ 5k iter 完整收敛 (reward +40.90, 比 LAFAN1 walk1 还好)</h3></summary>

> **背景**: smoke train 证 schema OK, 但不证 reward shaping 在 friend BABEL data 上能收敛。LAFAN1 walk1 之前在 BeyondMimic 训了 5k → +32.47, 现在看 friend 数据能不能至少同样。
>
> **操作**:
> ```bash
> cd third_party/RoobotMimc/whole_body_tracking
> python scripts/rsl_rl/train.py --task Tracking-Flat-G1-v0 \
>   --motion_file $(pwd)/tmp/seg_04401_g1.npz \
>   --num_envs 2048 --max_iterations 5000 \
>   --logger wandb --log_project_name roobotmimic_friend_walk \
>   --run_name seg_04401_5k_2048env_2026-05-13 \
>   --headless --device cuda:0
> ```
>
> **结果** (vs LAFAN1 walk1 BeyondMimic 5k):
> | 指标 | LAFAN1 walk1 (BeyondMimic) | Friend seg_04401 (RoobotMimic) | Δ |
> |---|---|---|---|
> | Mean reward (final) | +32.47 | **+40.90** | **+26%** |
> | error_anchor_pos | 0.71 m | **0.14 m** | **−80%** |
> | ee fall rate | 2.6% | **0.34%** | **−87%** |
> | Iter time | 1.06s | 0.79s | −25% |
> | 训练时长 | ~1h 47min | ~1h 5min | −39% |
>
> **分析**: friend BABEL data 全部 metric 都比 LAFAN1 walk1 好。可能原因:
> 1. Segment 短 (3.5s vs LAFAN1 walk1 ~260s) → 训练 budget 集中
> 2. BABEL motion 质量高 (AMASS retarget,有 source provenance)
> 3. fps 20 → 50 upsample 让运动更平滑,reduce velocity tracking error
>
> 10 个 ckpt 落地 `logs/rsl_rl/g1_flat/2026-05-13_13-22-22_seg_04401_5k_2048env_2026-05-13/` (model_500.pt → model_4999.pt)。
>
> **Wandb run**: https://wandb.ai/lingfanb-university-college-london-ucl-/roobotmimic_friend_walk/runs/91641f1i

> **决定 / Next**: friend data + RoobotMimic pipeline 完全可信。可以放心规划下一步(VAD wiring / 批量转换 / D 维度)。
