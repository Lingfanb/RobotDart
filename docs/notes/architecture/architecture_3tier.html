Build order: bottom-up

Tier 1 first → Tier 2 dispatcher → Tier 3 ACP. Each tier independently validated before composing upward.

Theoretical grounding · dual-process social cognition

ACP = System 2 deliberative (slow, goal-directed social planning, Kahneman 2011 / Strack & Deutsch 2004). VAD = System 1 reactive (fast, automatic affective realization). Mirrors human cognitive control.

Authorship + RAL relationship

Builds on prior in-lab work: V-A conditioned motion generation on G1 [Undergrad et al., RAL 2026, Lingfan = 2nd author]. The NMI paper extends that foundation with: (1) Dominance dim · (2) physical (handover) channel · (3) closed-loop deployment · (4) N=30 cross-skill user study · (5) 3-tier ACP→VAD→skill-bank framework.

Hero claim

First humanoid delivering nuanced expressive interaction across non-physical (gesture) and physical (handover) channels via unified social-variable control. Cross-channel consistency — same ACP target produces perceptually consistent affect across both channels — is the NMI empirical hero finding.
