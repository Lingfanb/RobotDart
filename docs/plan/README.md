# Plan · Goals & Roadmap

目标与执行路线图的统一入口。和其他目录的分工：

| 目录 | 干什么 |
|---|---|
| `plan/` | **我想去哪里** —— short/long-term goals, weekly plans, milestones |
| `notes/` | **怎么去** —— paper plan, architecture design, module build list |
| `knowledge/` | **已知的事** —— dataset/method/tool reference |
| `logs/` | **去了哪里** —— 每日工作日志 |
| `LOG_README.md` | TODO + 完成记录（active state） |

## 文件

| 文件 | 频率 | 内容 |
|---|---|---|
| [long_term.md](long_term.md) | 几月一更 | 1-3 年 vision，paper 提交后的下一步 |
| [short_term.md](short_term.md) | 每周一更 | 本周 focus + 下周 plan |
| [milestones.md](milestones.md) | 按需 | NMI 13 周里程碑表 |
| [risks.md](risks.md) | 按需 | 风险登记 + 应对 |
